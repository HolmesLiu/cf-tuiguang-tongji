## 💡 决策记录

| 时间 | 决策 | 原因 |
|---|---|---|
| 2026-06-08 | 不做 OAuth 授权流程 | 钉钉"待办"接口（topapi/workrecord/add）只需企业 corp access_token，不需要用户级 OAuth |
| 2026-06-08 | 任务删除采用「软删 + 归档」混合 | 兼顾可恢复性与可彻底清除 |
| 2026-06-08 | 详单渠道简化为 referer | 不为每个渠道建独立短链，保持轻量 |
| 2026-06-08 | 全静态资源走 Workers Static Assets | 一体化部署，最低成本 |
| 2026-06-08 | 域名 tuiguang.7kk.top | 大宇指定 |
| 2026-06-08 | 砍去 OAuth 授权流程 | 钉钉"待办"只需 corp access_token，不需要用户 OAuth |
| 2026-06-08 | 不选 Redux / Vue / React | 后台简单，原生 JS + CSS 够用 |
| 2026-06-08 | 不选第三方二维码服务 | qrcode npm 包 + R2 存储零成本 |
| 2026-06-08 | 不选 Redis | KV 够用 |
| 2026-06-08 | 不选 React SPA | 后台简单，原生 JS + fetch 即可 |
| 2026-06-08 | 用拼音命名 GitHub 仓库 | GitHub 不支持中文仓库名 |
