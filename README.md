# cf推广统计

> 基于 Cloudflare Worker 的企业推广监控平台

[![Version](https://img.shields.io/badge/version-v0.1.0-blue)]()
[![License](https://img.shields.io/badge/license-MIT-green)]()
[![Cloudflare](https://img.shields.io/badge/cloudflare-worker-orange)]()

---

## ✨ 特性

- 🎯 **任务发布**：操作人通过对话（agent）或后台直接发起推广任务
- 🔗 **专属短链 + 二维码**：每个推广人一份
- 📱 **钉钉深度集成**：自动同步通讯录 / 自动推送工作通知或待办
- 📊 **效果监控**：以独立 IP 为有效参数，详单可查
- 🌐 **Agent 友好**：提供 RESTful API，可被 agent（娜娜子）调用
- 💰 **最低成本**：完全运行在 Cloudflare 免费额度内
